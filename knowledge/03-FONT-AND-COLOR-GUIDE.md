# FONT & COLOR GUIDE (Knowledge File)

Goal: names must be **bold, large, and crystal clear** — AND stylish, with each
name's font/color matching the theme and the person's signature color.
Never use the same font/color for two names in one poster.

---

## DESCRIBE FONTS IN WORDS (image AIs respond to descriptions, not font names)
Use phrases like these:
- "bold metallic-chrome font with electric-blue glow"
- "ornate crimson gothic serif with red glow"
- "cracked blood-red horror font with dark shadow"
- "fun graffiti font, pink-and-blue with white outline"
- "royal gold serif with soft glow"
- "rugged carved-stone font with orange glow"
- "glowing neon-blue digital techno font"
- "elegant emerald-green serif with golden glow"
- "bold steel-silver font with red rim glow"
- "playful rounded bold font, warm yellow"

Always add: **"large, bold, crystal clear and perfectly readable"** to fight blur.

---

## SIGNATURE COLOR IDEAS (pick what fits; ensure contrast between names)
- Messi → sky-blue / white
- Ronaldo → red / silver
- Neymar → green-yellow / gold
- Kohli → orange / royal blue
- Dhoni → steel-blue / white
- Rohit → bronze / blue
- Hulk / nature → green
- Vampire / horror → crimson / blood-red
- Tech / futuristic → neon-blue / cyan
- Fire / lava themes → orange / molten red
- Ice / frost themes → icy-blue / white
- Royal / GOAT → gold
- Joker → green / purple
- Spider-Man → red / blue
- Venom → black / silver

If a person has no obvious color, choose one that CONTRASTS with the other
name(s) in the poster so both pop.

---

## READABILITY RULES (always apply)
1. Name text sits at the very bottom, large, occupying clear space.
2. Add a subtle outline or shadow so text stays readable over busy backgrounds.
3. Keep names in CAPS or Title Case — never tiny or thin.
4. If two themes/colors clash, default to: one warm color + one cool color.
5. For 3-character posters, give three clearly different colors (e.g. gold,
   silver, bronze OR blue, red, green).

---

## REACTION ICON STYLE
- Render the Facebook reaction icons as clean, recognizable buttons:
  - ❤️ Love = red rounded heart button.
  - 👍 Like = blue rounded thumbs-up button.
  - 😮 Wow = yellow round face with open mouth.
- Place each icon neatly between that person's character layer and real-self layer.
- Make icons glow softly to match the theme but stay instantly recognizable.
